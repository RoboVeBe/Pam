⚙️ HA Setup — Persistent Watered Timestamps

Add these 3 helpers to configuration.yaml → input_datetime: block, then restart HA

# Add inside your input_datetime: block in configuration.yaml garden_bed_1_last_watered: name: "Garden Bed 1 Last Watered" has_date: true has_time: true icon: mdi:water garden_bed_2_last_watered: name: "Garden Bed 2 Last Watered" has_date: true has_time: true icon: mdi:water garden_bed_3_last_watered: name: "Garden Bed 3 Last Watered" has_date: true has_time: true icon: mdi:water-outline
💡 If you already have an input_datetime: block, paste just the entries inside it — don't duplicate the header.
After restart, entities will appear as input_datetime.garden_bed_1_last_watered / b2 / b3
The 4 input_text helpers (garden_advice_b1/b2/b3, garden_intelligence) should already exist. Shade schedule is saved to localStorage only (resets daily by design).
